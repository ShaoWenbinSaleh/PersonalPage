Ring Ring Zoo

creative puzzle game with illustration style artwork

Animals are missing! Now let us catch animals back to the zoo! Draw a closed square or polygon fence by the stumps to square animals quickly in a grid-based game scene. The smaller the area of a polygon is, the higher the score you can get. In addition, different animals have different properties, so the score you get varies based on the types of animals you “caught”. Furthermore, there are some game items. You will fail to pass the current level if you cannot reach the goal score within the limited time. After clearing all the game levels, the “endless mode” will be unlocked which you can challenge yourself to get the highest scores you can in endlessly generated levels.

Voilent Loli

a story tells how terrible if one little girl gets angry

This game tells a story that one little girl distories buildings in order to save her brother. In this game, player can command the hero to move via different touch gesture. Tapping the screen means the attack commond. This hero can jump between these different buildings. There are also some powerful skills such as the skill showed in the video which can destory all the floor once. Item system has also been implemented. There will be a boss in some specific levels.All developed myself except some graphics work. Finished in Feb 2014.

Wiki Ninjia

real-time web education game inspired by Fruit Ninjia

This is one online web game. The front-end is developed by JavaScript and CSS(the motion animations are all CSS animations) and the back-end is developed by Express(a framework of Node.js).This game aims to benefit the learning experice of wikipedia. Players can see their real-time ranking. Missed items will appear to the left side and linked to the corresponding page of wikipedia. This game utilized the WikiMedia API to get the data from wikipedia. All developed by myself. Finished in May 2015.
